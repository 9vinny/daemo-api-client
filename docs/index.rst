Getting started
---------------

We recommend installing via ``pip``:

.. code-block:: bash

    $ pip install --upgrade daemo-api-client

You can also install from source:

.. code-block:: bash

    $ git clone https://github.com/crowdresearch/daemo-api-client
    $ cd daemo-api-client
    $ python setup.py install
